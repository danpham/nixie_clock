void uart_init ( void  )

Initialize the UART peripheral.

Configures the UART port, installs the UART driver, creates a mutex for thread-safe transmission, and sets up an RX queue.

UART parameters:

  • Baud rate: UART_BAUD_RATE
  • Data bits: 8
  • Parity: Disabled
  • Stop bits: 1
  • Flow control: Disabled
Returns
None
