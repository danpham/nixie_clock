Change the STA Wi-Fi credentials and reconnect.

Disconnects the current STA, applies a new SSID/password, and reconnects using the updated configuration. Does not affect AP mode.

Parameters
sta_ssidNew STA SSID (null-terminated)
sta_passphraseNew STA password (null-terminated)
Returns
ESP_OK on success, or an ESP-IDF error code.
