void wifi_callback ( void  )

Handles Wi-Fi initialization and configuration updates.

This callback retrieves the latest Wi-Fi configuration and either initializes the Wi-Fi (AP+STA mode) or updates the STA connection if already initialized. Ensures thread-safe access to the configuration using a mutex.

Note
Logs an error if updating the STA Wi-Fi fails.
