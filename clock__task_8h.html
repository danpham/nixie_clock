◆ clock_task_start()

void clock_task_start ( void  )

Start the clock task and initialize the update queue.

Creates the FreeRTOS queue clockUpdateQueue for receiving clock updates (from NTP or other tasks), then creates the clock_task FreeRTOS task.

Note
If the queue creation fails, the task is not started.

Update clock state from configuration.

Reads the latest configuration and updates the clock time if NTP is disabled. Intended to be triggered by configuration change events (boot, webserver update, etc.).
