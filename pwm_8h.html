void pwm_init ( void  )

Initialize the PWM peripheral.

This function configures the LEDC timer and channel to generate a PWM signal on the pin defined by PWM_GPIO.

Note
The PWM frequency is defined by PWM_FREQ_HZ.
Uses timer LEDC_TIMER_0 and channel LEDC_CHANNEL_0.
