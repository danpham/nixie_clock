void hv5622_init ( void  )

Initialize the HV5622 shift register.

Sets up SPI communication, configures the LE control pin, and sets its initial state. BL is managed by PWM.

Send 64-bit data to the HV5622 shift register.

The data is inverted before transmission. After sending via SPI, the LE pin is toggled to update the outputs.
